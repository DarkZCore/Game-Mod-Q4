To play this mod you have to go into your quake 4 main folder from steam and add the mod folder into its main directory. There are 2 ways in which this can be achieved download the pak001.pak from here and extract it into your desktop rename it to what you would like and then place it on the main directory to view the mod in the app. The next thing you can do is take the pak file and add it to q4base and then rename it to be the latest pak. Then open console by pressing "control + alt + ~" and type "map game/tutorial" then try and survive as long as you can.

Install Pak: https://drive.google.com/file/d/1IOBdX1fqP_FH45CztW_4q3_cGqbDsaA7/view?usp=sharing

Features:
Survive endless wave of monsters coming your way! They should explode after you die.

Upcoming 5 ki based attacks coming soon!

Melee attack system coming soon!

Find find the 7 dragon balls to escape onslaught of enemies. (Currently you can only pick them up)

Pick up different power ups along the way to help you with your journey.
Timemachine - allows you to fly!
Senzubean - heals you!
Armor - Gives you extra shield!
